# Быстрый старт: Запуск бота с Ollama

## Что сейчас происходит?

Бот работает **без нейросети** и выдает только шаблонные ответы.

Чтобы бот мог вести естественный диалог, нужно установить **Ollama** (бесплатную локальную нейросеть).

---

## Быстрая установка (5 минут)

### 1. Скачать Ollama

```
https://ollama.com/download
```

Нажмите "Download for Windows" → Запустите `OllamaSetup.exe` → "Install"

### 2. Скачать модель

Откройте PowerShell:

```powershell
ollama pull llama3.2:3b
```

Подождите 5-10 минут (~2GB).

### 3. Проверить

```powershell
ollama list
```

Должно быть:
```
NAME            SIZE
llama3.2:3b     2.0 GB
```

### 4. Перезапустить бота

```powershell
cd c:\BOT
python -m src.main
```

Должно быть:
```
[OK] Ollama is available (model: llama3.2:3b)  ✅
```

Вместо:
```
[WARNING] Ollama is not available  ❌
```

## Тестирование

Откройте бота в Telegram и попробуйте:

```
Вы: Привет, как дела?
Бот: Привет! У меня всё отлично! Чем могу помочь?  ✅

Вы: Расскажи как день прошел
Бот: День был хороший! А у вас? Чем помочь?  ✅

Вы: Какие услуги вы предоставляете?
Бот: [Подробный ответ об услугах]  ✅
```

## Если что-то пошло не так

### Ollama не запускается?

```powershell
net start ollama
```

### Модель не скачалась?

```powershell
ollama pull llama3.2:3b
```

### Бот всё еще выдает шаблоны?

1. Проверьте что Ollama работает:
   ```powershell
   ollama run llama3.2:3b "Привет"
   ```

2. Перезапустите бота:
   ```powershell
   cd c:\BOT
   python -m src.main
   ```

3. Смотрите логи - должно быть `[OK] Ollama is available`
